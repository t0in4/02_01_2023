Только один параметр больше нуля

Напишите программу, которая читает три числа и выводит true, только  
если ОДНО число больше нуля (т.е. > 0)  

В любом другом случае, выводит false  

Sample Input 1:  

1 1 1  

Sample Output 1:  

false  

Sample Input 2:  

1 0 -1  

Sample Output 2:  

true  

Sample Input 3:  

1 1 0  

Sample Output 3:  

false  

Sample Input 4:  

0 2 0  

Sample Output 4:  

true
